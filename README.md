What is it?
===========

A Puppet module that manages Redis.

Released under the Apache 2.0 licence

Usage:
------

To install:
<pre>
  include redis
</pre>

Lots of config parameters: [init.pp](https://github.com/evenup/evenup-redis/blob/master/manifests/init.pp)
